Chat AI - Asisten Penjualan

Tanya seputar data penjualan: dashboard, produk terlaris, laporan harian, perbandingan penjualan.

Input Data Stok BIZ

Silakan Copy (Salin) baris data dari file Excel/export BIZ, lalu Paste (Tempel) di dalam kotak di bawah ini. Data lama akan diganti dengan data baru.

Perhatian:

  1. Copy (Salin) data dari sheet BIZ/file export BIZ (header sudah termasuk).
  2. Paste (Tempel) ke dalam kotak di bawah ini.
  3. Klik tombol Proses & Preview Data untuk melihat pratinjau.
  4. Klik Simpan Data BIZ untuk menimpa data BIZ yang ada.

Input Data Stok Excel

Silakan Copy (Salin) data stok dari kolom B sampai W (nilai stok saja, tanpa kolom A/Cabang), lalu Paste (Tempel) di kotak bawah. Data lama akan diganti sesuai urutan baris.

Perhatian:

  1. Copy data dari kolom B sampai W (nilai stok produk) di file Excel Anda.
  2. Urutan baris harus sama persis dengan urutan cabang di sheet Update-STOCK.
  3. Paste (Tempel) ke dalam kotak di bawah ini.
  4. Klik tombol Proses & Preview Data.
  5. Klik Simpan Data untuk menimpa nilai stok yang ada.

Input Data Penjualan

Silakan Copy (Salin) baris data dari file Excel Laporan Penjualan Anda, lalu Paste (Tempel) di dalam kotak di bawah ini.

Panduan Input Data:

  1. Pastikan Anda telah meng-Unhide (tampilkan semua) kolom di Excel Laporan Penjualan WHO Anda.
  2. Pilih Cabang Anda dan Tipe Laporan (MST & STK / TsiApps / TsiEmployee) pada pilihan di bawah.
  3. Block dan Copy (Salin) baris data dari Kolom B sampai AF (Kolom Tanggal sampai JUMLAH).
  4. Paste (Tempel) ke dalam kotak di bawah ini, lalu klik tombol Proses & Preview Data.

Input Penerimaan (Barang Masuk)

Silakan Copy (Salin) baris data dari file Excel Surat Jalan / Penerimaan Anda, lalu Paste (Tempel) di dalam kotak di bawah ini.

Panduan Input Data:

  1. Pastikan Anda telah meng-Unhide (tampilkan semua) kolom di Excel Surat Jalan / Penerimaan Anda.
  2. Pilih Cabang Anda pada menu pilihan di bawah.
  3. Block dan Copy (Salin) baris data dari Kolom B sampai AF (Kolom Tanggal sampai JUMLAH).
  4. Paste (Tempel) ke dalam kotak di bawah ini, lalu klik tombol Proses & Preview Data.

Input Penerimaan dari Pabrik

Pilih Gudang, atur Tanggal (otomatis hari ini), lalu paste Jumlah dari Excel.

Cara Input:

  1. Pilih Gudang tujuan.
  2. Tanggal otomatis terisi hari ini, bisa diubah jika perlu.
  3. Copy kolom Jumlah dari Excel, paste ke kotak di bawah.
  4. Atau paste TANGGAL + Jumlah (2 kolom) jika tanggal per baris berbeda.

Input Distribusi

Pilih Gudang & Cabang, atur Tanggal (otomatis hari ini), lalu paste data dari Excel.

Cara Input:

  1. Pilih Gudang tujuan dan Cabang tujuan.
  2. Tanggal otomatis terisi hari ini, bisa diubah jika perlu.
  3. Copy kolom Jumlah dari Excel, paste ke kotak di bawah.
  4. Atau paste TANGGAL + Jumlah (2 kolom) jika tanggal per baris berbeda.
